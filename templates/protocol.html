{% if stage == 1 %}
Concept: Functional Form Selection.

Before testing, we must define the linear relationship. OLS assumes that the relationship between X and Y is linear in parameters.

> H0: No relationship exists.
> HA: A linear relationship exists.
{% endif %} {% if stage == 2 %}
Concept: Diagnostic Health.

We analyze the residuals (errors) of the model. If errors are correlated or variance is not constant, standard errors are invalid.

Breusch-Pagan: Tests for constant variance.
> H0: Homoskedasticity.

Breusch-Godfrey: Tests for serial correlation.
> H0: No autocorrelation.

{% endif %} {% if stage == 3 %}
Concept: Statistical Significance.

We use T-tests to determine if individual variables have a meaningful impact on the target.

> H0: Coefficient = 0 (No effect)

> HA: Coefficient ≠ 0 (Real effect)

If P-Value < 0.05, we reject H0. The variable is statistically significant.

{% endif %} {% if stage == 4 %}
Concept: Goodness of Fit.

How much of the variance in our target vector is actually explained by our predictors?

R-Squared: % of explained variance. Higher is better, but it always increases with more variables.

Adjusted R-Squared: Penalizes you for adding useless variables. The true measure of fit.

{% endif %}
